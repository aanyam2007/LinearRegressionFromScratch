# Linear Regression From Scratch(Python)

This project implements **Linear Regression using Gradient Descent from scratch** without using any machine learning libraries such as scikit-learn.

## Features
- Mean Squared Error loss
- Gradient Descent optimization
- Clean, modular code

## Mathematical Formulation
ŷ = Xw + b

Loss Function:
MSE = (1/n) Σ(ŷ − y)²

Gradients:
dw = (1/n) Xᵀ(ŷ − y)  
db = (1/n) Σ(ŷ − y)
